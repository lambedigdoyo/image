Canvas: The lineWidth Property

Drawing Straight Lines - HTML5 Canvas

Your browser does not support Canvas. 
<canvas id='cvs' width='200' height='150'></canvas>

<script>
var canvas = document.getElementById("cvs");
var ct = canvas.getContext("2d");
ct.beginPath(); // start-drawing: new Path
ct.moveTo(190, 10); // start point (x1, y1)
ct.lineTo(10, 140); // end point (x2,y2)
ct.lineWidth = 10; // defines the line width to use
ct.stroke(); //draw (x1, y1) to (x2, y2)
</script>
